# Data Visualization Backgrounds

Generate backgrounds and decorative elements for data dashboards and infographics.

## Quick Generate

```bash
# Dashboard background
fairstack image "Abstract data visualization background, flowing lines and nodes, dark navy with teal accents, modern analytics aesthetic, subtle grid pattern" \
  --model seedream-4.5-t2i --aspect-ratio landscape_16_9

# Infographic element
fairstack image "Clean isometric illustration of data flow, connected blocks and arrows, blue and teal palette, modern flat style, white background" \
  --model ideogram-v3-t2i
```
