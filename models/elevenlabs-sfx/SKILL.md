# ElevenLabs SFX

Generate any sound effect from a text description.

## Quick Start

```bash
fairstack voice "Thunderstorm with heavy rain" --model elevenlabs-sfx
```

## Pricing and Parameters

```bash
fairstack models --detail elevenlabs-sfx  # Current pricing, parameters, capabilities
```
